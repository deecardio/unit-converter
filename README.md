# Unit Converter 🔄
A Python console app that converts temperature, distance and weight.

## Features
1. Temperature: Celsius ↔ Fahrenheit
2. Distance: Kilometers ↔ Miles
3. Weight: Kilograms ↔ Pounds

## How to use
1. Run the program
2. Choose the conversion type from the menu (1-6)
3. Enter the number you want to convert
4. Get the result

## Error handling
1. Invalid menu choice → program asks again
2. Invalid number input → program asks again
3. Enter 0 to exit

## Technologies
- Python 3
